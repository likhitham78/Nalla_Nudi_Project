📘 ನಲ್ಲ-ನುಡಿ (Nalla-Nudi)

An offline bridge dictionary app designed to help Kannada-medium students easily understand English technical terms in their native language.

🌟 Overview

Nalla-Nudi is a lightweight Android application that provides fast, offline access to technical terms with Kannada meanings. It is built to support students in Science, Mathematics, and Commerce backgrounds.

✨ Key Features
⚡ Ultra-fast offline search (under 200ms)
📖 English → Kannada bilingual dictionary
🔊 Built-in Text-to-Speech (TTS) pronunciation
📚 Subject-wise filtering (Science, Math, Commerce)
⭐ Save words to My List
🧠 Flashcard-based revision system
🔌 Fully offline (no internet required)
🏗️ Tech Stack
💻 Kotlin (Android Development)
🎨 Jetpack Compose / XML UI
🗄️ Room Database (FTS4 search optimization)
🧩 MVVM Architecture
⚙️ Hilt Dependency Injection
🔊 Android Text-to-Speech API
📱 App Modules
🏠 Home – Search and explore terms
📄 Detail Page – Full meaning with pronunciation
⭐ My List – Saved important words
🧠 Flashcards – Revision mode for learning
🚀 Why This Project?

This project was built to:

Bridge the language gap in technical education
Help Kannada-medium students learn faster
Provide offline learning support without internet dependency
